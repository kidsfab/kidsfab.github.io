Ровер

На задней панели модуля управления имеются отверстия для установки на основание макетной платы “Знаток”. Соединение выводов модуля управления и элементов схемы может осуществляться стандартными коннекторами конструктора “Знаток”.

Цифровые входы/выходы: D3,D5,D6,D9 - обеспечивают два уровня напряжения 0 или 5В; Могут использоваться как выводы с широтно-импульсной модуляцией (0-255).

Аналоговые входы/выходы: A2,A5 - используются для считывания напряжения с аналоговых датчиков (датчик света, датчик звука, датчик касания и.т.д). Выдают значение напряжения в градации от 0 - 1023, что по умолчанию соответствует 0 - 5 В.

5V: данный вывод постоянно (не зависимо от программы) обеспечивает напряжением 5В и может быть использован для питания датчиков.

GND и VIN: могут быть использованы для питания Arduino_nano батарейными блоками конструктора “Знаток”. В режиме программирования (подключение к компьютеру) питание осуществляется через USB кабель.

Внимание! Не рекомендуется подавать более 50 мА нагрузки (подключать моторы постоянного тока, сервоприводы) на выводы(пины) модуля управления т.к. это может привести к неисправности Arduino_nano!

Программирование в Snap4Arduino

Программирование платы управления Arduino_mind на Snap4Arduino осуществляется при помощи библиотеки блоков znatok_block.xml, которая содержит следующие блоки:

Включить ПИН № - осуществляет включение/выключение цифровых пинов. В качестве входных аргументов необходимо ввести номер пина (3,5,6 или 9) и установить режим работы “Истина/Ложь” => что соответствует подачи на пин 5 В либо 0.

Напряжение на ПИН № - данный блок используется для подачи сигнала с ШИМ на цифровой пин. Входные аргументы: номер пина на который требуется подать сигнал с ШИМ (3,5,6 или 9) и параметр сигнала ШИМ (от 0 до 255).

Значение ПИН № - блок предназначен для считывания напряжения с аналогового вывода. В качестве входного аргумента указывается номер аналогового вывода с которого необходимо считать напряжение (2 или 5).

Пример: При нажатии на клавиатуре клавиши ”a” на плате Arduino_nano загорается светодиод. При нажатии клавиши ”f” светодиод гаснет.
